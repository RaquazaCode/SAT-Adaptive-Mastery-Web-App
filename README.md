# SAT Adaptive Mastery Web App

Adaptive SAT training system that models the test, not just content. Tracks
skills at the question-type level, estimates ability, and schedules practice
based on accuracy, errors, and time behavior.

## Getting Started

1. Install dependencies: `npm install`
2. Run the dev server: `npm run dev`
3. Lint the code: `npm run lint`

## Project Structure

- `src/` application source
- `prd/` core product specification
- `research/` deep research inputs and analysis
- `reference/` visual references and template exports
